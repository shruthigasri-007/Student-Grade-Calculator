# 📖 Project Overview

The **Student Grade Calculator** is a Java console application that calculates the overall academic performance of a student.

The user enters the number of subjects and marks obtained in each subject. The application automatically calculates:

- ✅ Total Marks
- ✅ Average Percentage
- ✅ Final Grade

This project demonstrates the use of loops, conditional statements, user input handling, and arithmetic operations in Java.
